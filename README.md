# test2

A Next.js application with API routes for saving data.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Running the Application

```bash
# Development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The application will be available at `http://localhost:3000`

## Project Structure

- `/pages` - Next.js pages and API routes
- `/pages/api` - API endpoints
- `/public` - Static files

## API Endpoints

- `POST /api/save-data` - Save data endpoint
